    4 Hi week 4

    We analyzed the hardware design for our parallel bubble sort, and here were some of those numbers.

    • Case where N=P

      • Last processor’s first output = 2P+1

      • First Processor is done(computing, but still may be holding number in out register) = N+1

      • When is the last processor done= 3N == 3P

    • Case where N > P... introduce K = N/P

      • First Output = 2P+1

      • Last Processor done = 2P+N

    • Putting it all together (+ or - 1)

      • Cycles for ith processor done = N2/2P + N/2 +2i

      • Cycles for last output appear = N2/2P + N/2 +2P +1
